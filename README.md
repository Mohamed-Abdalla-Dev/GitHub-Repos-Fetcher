# 🐙 GitHub Repos Fetcher

A simple application that fetches GitHub repositories using a username.  
It displays repository names with links to visit them.

## 🚀 Features

- Fetch repositories using GitHub API  
- Display repository names  
- Direct links to visit each repository  
- Simple and clean UI  

## 🛠 Technologies Used

- HTML  
- CSS  
- JavaScript(Vanilla)  
- GitHub API  

## 📌 Goal

Practice working with APIs, fetch, and displaying dynamic data.
